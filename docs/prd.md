# LinkPilot 插件产品需求文档（PRD）

---

## 一、产品概述

### 1.1 产品名称

**LinkPilot**

### 1.2 产品目标

LinkPilot 是一款专为 SEO 从业者设计的浏览器插件，旨在提升反向链接建设（Backlink Building）的效率与系统化程度。它覆盖从外链采集、分类管理，到自动化提交等完整工作流，并支持外链质量评估、任务追踪、评论生成等扩展功能。

### 1.3 用户群体

* SEO 从业人员
* 独立站网站主、自媒体内容运营者
* 内容营销团队

### 1.4 产品定位

LinkPilot 是一个“半自动化反向链接获取助手”，强调人机协作下的效率最大化，致力于解决当前手动搜集、管理和提交外链效率低、可追踪性差的问题。

---

## 二、核心功能与模块

### 2.1 外链采集模块

* 从 Ahrefs 页面中自动提取外链（基于 DOM 抓取）
* 从任意网页中提取页面上的所有外链（非本站链接）
* 支持通过 CSV 文件导入外链数据
* 本地以 JSON 文件结构保存采集结果
* 自动识别并标记 DoFollow / NoFollow 状态
* 可配置站点白名单、导入清洗逻辑（去重、格式验证）

### 2.2 外链管理模块

* 支持对外链数据进行增删改查（CRUD）
* 每条外链支持类型标注（博客 / 论坛 / 聚合站 / 付费等）
* 支持手动标记所属行业、领域标签、是否可投稿
* 自动识别目标页面平台（如 WordPress、Disqus 等）
* 支持按类型、状态、标签、来源等维度筛选
* 导出功能：JSON / CSV

### 2.3 外链提交模块

* 配置提交站点信息（主站链接、品牌名、行业标签等）
* 外链清单可作为提交任务清单（支持多站点交错管理）
* 插件访问外链页面并自动识别评论区或投稿入口
* 调用大语言模型生成评论内容（可选风格）
* 自动填充、自动提交，并记录状态
* 提交结果标记成功 / 失败（并记录失败原因）

---

## 三、用户流程

1. 用户配置站点信息
2. 在 Ahrefs 或任意页面中点击插件，采集外链
3. 浏览和标记外链类型、状态、平台等
4. 将特定外链加入提交清单
5. 插件逐个访问并尝试提交（生成评论 / 填写表单）
6. 成功或失败结果更新状态并记录日志

## 四、技术架构与权限说明

### 4.1 插件结构

* MV3 架构：使用 content script 与 background 交互
* UI 使用 popup 页面和设置页面进行交互
* 数据存储：本地 JSON 文件，未来可切换为远程数据库或 Notion API

### 4.2 插件权限

插件需申请以下权限以支持目标功能：

* `<all_urls>`：访问并操作任意外链页面 DOM（提交入口识别与提交）
* `scripting`：动态注入内容脚本以控制页面行为
* `storage`：保存本地配置、站点信息与外链数据

> ⚠️ 请注意：在发布 Chrome 插件时，应限制 `<all_urls>` 为特定外链来源域名，以避免审核风险。

## 五、后续扩展方向（Roadmap 简要）

* 支持远程数据同步（如 Supabase）
* 支持团队协作与任务分发
* 表单与评论提交规则学习与复用（自动适配结构）
* 更智能的链接价值评估（基于权重/收录/标签等）
* Web 端管理后台（用于多端联动）
